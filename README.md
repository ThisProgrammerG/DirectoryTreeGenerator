# DirectoryTreeGenerator

Generate a directory tree with the current working directory or optionally pass in a different path.

```
python main.py
```

or 
```
python main.py "C:\path\to\your\folder\"
```

Outputs:
```
Directory/
└── file_dir.exe
├── A/
│   ├── file_a.py
│   └── file_b.py
├── B/
│   └── file_c.txt
│   └── D/
│   │   └── file_d.jpg
└── C/
│   └── E/
```
